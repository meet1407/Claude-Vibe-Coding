Machine Learning Fundamentals

Machine learning is a subset of artificial intelligence that enables systems to learn from data and improve their performance without being explicitly programmed. It uses statistical techniques to give computers the ability to learn patterns from data.

Types of Machine Learning:
1. Supervised Learning: The model learns from labeled training data. Examples include linear regression for predicting house prices, and classification algorithms for spam detection. Common algorithms are decision trees, support vector machines, and neural networks.

2. Unsupervised Learning: The model finds hidden patterns in unlabeled data. K-means clustering groups similar data points together. Principal Component Analysis (PCA) reduces dimensionality. These are useful for customer segmentation and anomaly detection.

3. Reinforcement Learning: An agent learns by interacting with an environment and receiving rewards or penalties. Used in game-playing AI (like AlphaGo), robotics, and recommendation systems.

Key Concepts:
- Training Data: The dataset used to teach the model.
- Features: Input variables used for prediction (e.g., age, income, location).
- Labels: The output variable the model is trying to predict.
- Overfitting: When a model memorizes training data but fails on new data.
- Underfitting: When a model is too simple to capture patterns in the data.
- Cross-Validation: Technique to evaluate model performance using multiple train/test splits.

Evaluation Metrics:
- Accuracy: Percentage of correct predictions.
- Precision and Recall: Used for imbalanced classification problems.
- Mean Squared Error (MSE): Common metric for regression tasks.
- F1 Score: Harmonic mean of precision and recall.
